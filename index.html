How does “Import participants” work?

Importing a list lets Crono show each runner's name next to their number — in the results table and in the exported CSV. It's optional: without it, timing still works, you just see numbers only. Importing again adds to or updates the existing list (it never deletes your recorded times).

Upload a .csv file with one participant per line. The columns are number, name, sex, birth year — in that order: 

number,name,sex,birth_year
12,Ana Popescu,F,1998
13,Mihai Ionescu,M,1986
27,Andrei Georgescu,M,1971
  • Sex and birth year are optional. Without them a runner still gets timed, but only shows up in the “All” ranking — not in the men/women or age-category rankings. You can fill them in later by tapping the small + cat chip next to a runner in the results.
  • sex = M or F (also accepts male/female, masculin/feminin).
  • birth_year = a 4-digit year; Crono derives the athletics age category (10-year brackets, e.g. M30–39).
  • A header row like number,name,sex,birth_year is detected and skipped.
  • Comma ,, semicolon ; and tab separators all work; wrap names containing a comma in quotes, e.g. "Popescu, Ana".
  • You can create the file in Excel / Google Sheets → Save as / Download → CSV.
